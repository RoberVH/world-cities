<h1>Visiting Cities</h1>

Code to learn and test Typescript

App was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

This App allows to search and display current weather conditions and webcams (if available) of cities in the world

It is hosted at https://visitingcities.netlify.app/

It requires to Sign In through a third-party method. The  user can click on one of four social media icons to grant permission.

No custom server App is intended to be developed and used with this App, It doesn't use a  backend server.
It uses Firebase to access third-partu authorization

No data other than email is store at FireBase
