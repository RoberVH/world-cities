App bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

App to look and display current weather conditions and webcams if available of cities in the world

It was made to test typescript on React

It is hosted at https://visitingcities.netlify.app/

There is not backend server

It requires to Sign In through a third-party method. The  user can click on one of four social media icons to grant permission.
No custom server App is intended to be developed and used with this App
It uses Firebase to access third-partu authorization

* No data other than email is store at FireBase
